# 나는 돈이 좋아~

경제뉴스를 제목, 언론사, 발행일, 짧은 요약, 원문 링크 중심으로 읽고 기록하는 React 프로토타입입니다. 기사 본문 전체를 저장하거나 표시하지 않는 구조를 기본 원칙으로 잡았습니다.

실시간 뉴스는 `나는돈이좋아.bat`, `나는돈이좋아-이미지버전.bat`, `나는돈이좋아__(r).bat`처럼 `.bat` 파일로 열 때 동작합니다. HTML만 더블클릭하면 RSS 서버가 꺼져 있을 수 있어 샘플 뉴스로 보일 수 있습니다.

## 열기

- 이미지 버전으로 보기: `나는돈이좋아-이미지버전.bat`
- 꽃게 이모티콘 버전으로 보기: `나는돈이좋아__(r).bat`
- 휴대폰에서 보기: `휴대폰으로보기.bat`
- 이미지 버전 HTML: `나는돈이좋아.html`
- 꽃게 이모티콘 버전 HTML: `나는돈이좋아__(r).html`
- 배포 전 실시간 확인: `실시간배포확인.bat`

## 나중에 보내거나 배포할 때

HTML 파일만 보내면 받는 사람 PC에서 RSS 서버가 없기 때문에 샘플 뉴스로 보일 수 있습니다. 실시간 뉴스까지 보이게 하려면 `dist` 화면과 `/api/news` 서버가 함께 있어야 합니다.

가장 쉬운 배포 방식은 Vercel입니다.

```text
Build Command: npm run build
Output Directory: dist
News API: /api/news
```

이미 `api/news.js`와 `vercel.json`을 넣어두었기 때문에 Vercel에 프로젝트 폴더를 올리면 사이트 주소 하나로 앱과 실시간 뉴스가 같이 동작합니다.

로컬에서 배포처럼 확인하려면 `실시간배포확인.bat`을 실행하세요. 이 파일은 `dist`를 빌드한 뒤 `http://127.0.0.1:5180`에서 화면과 `/api/news`를 함께 확인합니다.

## 들어간 기능

- 여러 언론사의 경제뉴스 카드 UI
- 금리, 환율, 주식, 부동산, 기업, 물가, 글로벌경제 카테고리 필터
- 제목과 요약 안의 경제 용어 자동 감지 및 하이라이트
- 50개 이상 경제 용어 사전, 쉬운 설명과 예시 문장
- 카테고리별 뉴스 읽기 가이드
- 내 생각, 한 줄 평, 태그 기록
- 저장한 뉴스와 메모를 모아보는 나의 경제뉴스 노트
- 클릭한 용어를 기록하는 오늘 배운 용어 위젯
- 설명을 보고 용어를 맞히는 4지선다/직접입력 게임
- RSS 연동을 위한 Express 프록시 서버 예시

## 실행

```powershell
npm install
npm run dev
```

현재 작업 환경에서는 번들 Node로 설치와 빌드를 확인했습니다. 개발 서버는 아래 주소로 실행 중입니다.

```text
http://127.0.0.1:5173
```

## PC와 휴대폰에서 쉽게 보기

PC에서만 볼 때는 `열기.bat`을 실행하면 됩니다.

휴대폰에서도 보려면 `휴대폰으로보기.bat`을 실행하세요. 창에 아래처럼 휴대폰 접속 주소가 표시됩니다.

```text
http://내-PC-IP주소:5173
```

휴대폰과 PC가 같은 와이파이에 연결되어 있어야 하며, Windows 방화벽 확인창이 뜨면 허용을 눌러야 합니다.

가장 편하게 누구나 접속하게 만들려면 GitHub Pages, Netlify, Vercel 같은 무료 웹 호스팅에 배포하면 됩니다.

## RSS 연동 예시

```powershell
npm run rss:server
```

프런트엔드의 `src/newsSources.js`에서 `fetchArticles()`를 `/api/news` 호출로 바꾸면 RSS 서버와 연결할 수 있습니다.

## 저작권 안전 체크리스트

- 기사 전문을 저장하거나 화면에 표시하지 않습니다.
- 외부 뉴스는 제목, 언론사, 날짜, 짧은 요약, 원문 링크만 정규화합니다.
- RSS의 `content:encoded` 같은 본문 필드는 사용하지 않습니다.
- 요약은 짧게 제한하고, 원문 보기 버튼은 언론사 기사 페이지로 연결합니다.
